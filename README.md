# Medical Health Companion

This is a local Streamlit application that acts as your personal medical assistant.

## Features
- **Profile Management**: Store allergies, medications, and history.
- **Consultation Log**: Keep track of doctor visits.
- **AI Assistant**: Chat with a Gemini-powered assistant that knows your medical context.

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the app:
   ```bash
   streamlit run app.py
   ```

3. Enter your Google Gemini API Key when prompted in the sidebar.
